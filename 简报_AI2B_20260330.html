🚨 重大突破 01 / 05
Anthropic · Claude

Claude Computer Use 正式上线 —— AI 从"对话助手"变为"数字员工"，直接操控 Mac 电脑

发布时间2026年3月23日（面向 Claude Pro / Max 用户，研究预览版） 产品名称Claude Computer Use（集成于 Claude Cowork） 核心能力Claude 可直接点击、打字、打开应用、浏览网页，完成用户指定的桌面任务 当前限制仅限 macOS，需 Claude Pro（$20/月）或 Max（$100-200/月）
Anthropic 正式推出 Computer Use，这是 AI 助手从"语言生成工具"进化为"具身数字员工"的里程碑。核心设计：连接器优先原则——Claude 优先调用 Slack、Gmail 等 API 连接器；无可用 API 时才切换屏幕操控模式。安全机制三层：① Prompt Injection 自动扫描（防恶意网页注入攻击）；② 所有鼠标/键盘操作实时可见；③ 每类新操作须用户明确授权。与 Dispatch（跨设备派任务）+ Scheduled Tasks（定时任务）联动，构成完整的企业级 AI 工作流自动化闭环。
💡 对 2B AIPC 的战略影响（P0 级）
  • 重新定义 AI PC：AI PC 不再是"本地算力强的设备"，而是"企业 Agent 可信本地执行端"——Computer Use 证明了这一价值主张的可行性
  • Windows/Linux 窗口期：Computer Use 目前仅支持 macOS，Windows 企业市场缺口巨大。谁能争取到 Anthropic Windows 优先适配权，谁就占据企业市场的关键差异化优势
  • 安全标准即入场券：Prompt Injection 防护 + 操作可见 + 明确授权已成行业基线要求，2B AIPC 若无同等安全架构则无法进入企业采购清单
信源：Anthropic 官方博客 · claude.com/blog · 2026-03-23
